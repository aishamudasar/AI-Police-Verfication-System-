# AI-Based Identity Verification System

This project is a prototype implementation of an AI-based identity verification system for online police complaint platforms.

## Features
- Face Detection (MediaPipe - CNN based)
- Face Recognition (Embedding-based)
- Liveness Detection (Blink detection)
- Identity Matching (Euclidean Distance)

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

## Research Purpose
This project supports research on enhancing online police complaint systems using AI-based biometric verification.
